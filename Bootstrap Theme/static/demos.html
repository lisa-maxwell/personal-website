Designed with customization & integration in mind.

As developers we know that implementation isn't easy, and scope-creep is inevitable. Bodeo was built with developers & designers in mind. The theme is modular structure making it easy to reuse any of the pages and components. The entire look-and-feel can be customized using CSS or SASS.

Get a quick start.

Kickstart your project with one of the many included starter examples. The package includes a minimal "empty" starter, full static site, templates with partials, or get started with the working AngularJS MVC app. You can also pick from code snippets that demonstrate various elements and content layouts such as navbars, forms, content sections, cards, lists, option tables and more. These snippets are designed to be singular in purpose, reuseable, and easy to modify. There are endless possibilities.
