Although many themes claim to be for "Bootstrap 4" they're only Bootstrap 4 compatible and are bloated with extraneous CSS and customizations. These themes don't actually make full use of Bootstrap 4's powerful responsive grid and flexbox components. Bodeo is intended for designers & developers that want a complete Bootstrap 4 ready solution with full breadth & depth of features. Eventhough this theme has a custom look-and-feel, special care is made to minimize CSS changes to utilize Bootstrap out-of-the-box. Additionally, the HTML markup is optimized for speed, and the included image assets are "Lossy" compressed using the method recommended by Google PageSpeed Insights.

  • Mobile-ready, 100% responsive design for all devices
  • HTML markup and CSS class use is optimized for speed
  • Lossy compressed images (jpg, png) included

About

Here's a custom Bootstrap 4 theme to give your next project a beautiful start. Bodeo isn't just Bootstrap 4 "compatible", it's completely built on Bootstrap 4's responsive design and flexbox components. This multi-purpose theme and the included templates are based on HTML5 & CSS3 standards to ensure consistency and cross-browser support. Bodeo is intended for designers & developers that want a complete Bootstrap 4 ready solution with full breadth & depth of features. The theme can be customized using CSS or SASS, and it's modular structure make it easy to reuse specific pages and components.

  • High-contrast color palette
  • Rokkitt sans-serif font
  • Big bold custom accent color
